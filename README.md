## 🔍 Sobre o Projeto

O portal **Oficina Cross Bike** foi desenvolvido para transpor a tradição da loja física para o ambiente digital. Ele serve como catálogo interativo de produtos e ponto de contato direto para serviços especializados de manutenção.

### 🎯 Objetivos Estratégicos
* **E-commerce de Performance:** Exibição clara de modelos (Aro 20, 24, 26, 29) com sistema de descontos via PIX.
* **Vitrine de Componentes:** Seção dedicada a peças críticas (quadros, correntes, mesas e segurança).
* **Conversão Direta:** Integração com WhatsApp para orçamentos rápidos.
